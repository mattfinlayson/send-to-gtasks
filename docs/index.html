Send to Google Tasks

You found something worth doing. Don't lose it in a tab. Save any page to Google Tasks in one click.

Add to Chrome — it's free View on GitHub
How it works
1
Install the extension

Add it to Chrome from the Web Store. Sign in with Google on first use.

2
Click the icon or press Ctrl+Shift+K

On Mac use ⌘+Shift+K. Optionally add notes or a due date.

3
Task saved

The page title and URL appear instantly in your Google Tasks list.

Features
  • ✓ One-click or keyboard shortcut task creation
  • ✓ Automatic page title and URL capture
  • ✓ Add notes and a due date before saving
  • ✓ Choose which Google Tasks list to save to
  • ✓ Quick save mode — skip the popup entirely
  • ✓ Badge indicator confirms success or error
  • ✓ Detects duplicate URLs before saving
Options

Right-click the extension icon and select Options to configure:

  • ✓ Set your default task list
  • ✓ Toggle Quick save mode to bypass the popup
  • ✓ Customize the keyboard shortcut via chrome://extensions/shortcuts
Permissions
Permission Why it's needed
activeTab Read the current page title and URL
storage Save your task list preference locally
identity Authenticate with your Google account
alarms Clear the badge indicator after a few seconds

This extension only communicates with the Google Tasks API. No data is sent anywhere else.
